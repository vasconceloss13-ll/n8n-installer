Gerador de Schema
n8n self-hosted

Gera as chaves de segurança automaticamente e monta o schema completo. Escolha o ambiente, preencha seu domínio e copie. Simples assim.

✦ Antes de começar — pré-requisitos

  1. VPS com EasyPanel instalado — um servidor rodando, com o EasyPanel ativo e acessível.
  2. Domínio próprio comprado — ex.: seudominio.com.br, em qualquer registrador (Registro.br, Cloudflare, GoDaddy…).
  3. DNS apontando pra VPS — crie registros A pros subdomínios apontando pro IP do servidor:
  4.   • agentes.seudominio.com.br → IP da VPS (editor)
  5.   • hooks.seudominio.com.br → IP da VPS (webhook)

✦ Antes de começar — pré-requisitos

  1. Docker + Docker Compose instalados — no Windows/Mac use o Docker Desktop; no Linux, o pacote docker + docker-compose-plugin.
  2. Porta 5678 livre — é a porta que o editor vai usar. Se já tiver algo rodando nela, dá conflito.
  3. ~2 GB de RAM livres — suficiente pro modo simples. O modo fila pede um pouco mais.
  4. Como subir: salve o arquivo como docker-compose.yml, abra o terminal na pasta dele e rode docker compose up -d. Depois acesse http://localhost:5678.
🔐

Guarde estas senhas antes de fechar a página. Elas são geradas no seu navegador e não ficam salvas em lugar nenhum — se você fechar a aba, somem. A N8N Encryption Key é a mais importante: ela criptografa as credenciais dos seus workflows e não pode ser alterada depois, senão você perde acesso a tudo que já salvou.
